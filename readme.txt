== Description ==

This plugin is used to show events in calendar. It is responsive event calendar. You can show all events according to date.
All events show in attractive way in calendar.
Demo of plugin - http://www.mummamart.com/wordpress-event


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/awesome-event-calendar` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add following shortcode in any page or post [event-calendar-aec]
